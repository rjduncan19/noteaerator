# Age-rating questionnaire answers (IARC)

Note Aerator is a local-only Markdown viewer. It has no online services, no
user-to-user features, no advertising, and no purchases. The honest answers
land it at an "All ages" / IARC 3+ / ESRB Everyone rating.

Use the following answers when filling out the Partner Center
"Age ratings" page.

## Common across all categories

- **Violence**: None.
- **Sexual content / nudity**: None.
- **Profanity / crude humor**: None.
- **Controlled substances** (alcohol, tobacco, drugs, gambling references):
  None.
- **Horror / fear**: None.
- **Discrimination**: None.

## Interactive elements

- **Users interact with each other** (chat, messaging, multiplayer): **No**.
- **User-generated content is shared with other users**: **No**.
  (Notes stay on the user's PC; nothing is uploaded by the app.)
- **Shares user's location with other users**: **No**.
- **Allows purchases of digital goods**: **No**.
- **In-game / in-app advertising**: **No**.

## Data collection

- **Collects or transmits personal information**: **No**.
- **Collects or transmits user location**: **No**.

## Miscellaneous

- **Unrestricted internet access** within the app: **No**.
  (Note Aerator renders local files in an embedded WebView2; it does not
  navigate to arbitrary URLs.)
- **References to real-world brands / products**: **No**.

Expected outcome: IARC 3+ / ESRB Everyone / PEGI 3 / USK 0 / ACB G.
